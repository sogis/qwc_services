Abschaltung EPSG Code 21781

Im Herbst 2016 wurde im Kanton Solothurn der Bezugsrahmenwechsel von LV03 (EPSG Code 21781) auf LV95 (EPSG Code 2056) durchgeführt. Der EPSG Code 21781 wurde während einer Übergangsphase von 5 Jahren vom AGI weiter unterstützt. Mit dem 31.3.2022 wird die Unterstützung durch das AGI endgültig eingestellt. Somit unterstützen zukünftig auch unsere Geodienste diese Projektion nicht mehr. Bitte verwenden Sie stattdessen den neuen Bezugsrahmen LV95 mit dem EPSG Code 2056.
